# Employee-Task-Manager
Simulates real-world task management between managers and employees

## Description
This project is a console-based Workflow Management System developed using Core Java. It simulates real-world task handling between managers and employees. Managers can create and assign tasks with priorities, while employees can view and update the status of their assigned tasks. The system uses multithreading to simulate multiple users and file handling to store task data persistently.

## Features

- Role-based Login (Manager & Employee)
- Task Creation and Assignment
- Task Priority Management (HIGH, MEDIUM, LOW)
- View Assigned Tasks
- Update Task Status (COMPLETED)
- View All Tasks (Manager)
- Multi-threaded User Simulation
- File-based Data Storage
